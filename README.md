# DTF Printer USA — Stafford TX Landing Pages Preview

Throwaway Next.js + Tailwind preview deployment of two cross-linked local landing pages drafted for DTF Printer USA's Shopify site. Built for client preview before Max pastes the final content into Shopify.

## Routes

- `/` — preview index
- `/dtf-supplies-near-stafford-tx` — supplies-focused landing
- `/dtf-printers-heat-presses-near-stafford-tx` — high-ticket equipment landing

## Local dev

```sh
npm install
npm run dev
```

Open http://localhost:3000.
